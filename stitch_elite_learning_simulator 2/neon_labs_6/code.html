实验编号: S-402

Memory Buffer Overflow

在本模拟中，你将探索内存管理的基本漏洞。目标是通过注入超长字符串触发缓冲区溢出，从而覆盖指令指针寄存器（EIP）。

target 实验目标

  • check_circle 识别目标函数 `vulnerable_service()` 中的输入点。
  • radio_button_unchecked 计算达到返回地址所需的偏移量。
  • radio_button_unchecked 成功重定向执行流至 `secret_callback()`。

warning 系统约束

缓冲区大小 64 Bytes
堆栈保护 Disabled
OS 架构 x86_64
