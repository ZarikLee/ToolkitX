变量与动态逻辑

Python 变量是用于存储值的保留内存位置。在我们的模拟环境中，我们将变量视为动态数据缓冲区。

timer 阅读时间: 12m
bolt EXP: +450

01. 变量赋值

Python 是动态类型的，这意味着你在创建变量时不需要声明其类型。解释器会根据赋给变量的值自动推断其类型。

main.py 
# 简单赋值
user_id = 42
system_status = "ACTIVE"
encryption_strength = 0.985

# 多重赋值协议
x, y, z = "Alpha", "Beta", "Gamma"

print(f"系统状态: {system_status} | 用户 ID: {user_id}")

02. 逻辑运算符

为了构建复杂的决策树，我们使用逻辑运算符：and（与）、or（或）和 not（非）。

布尔校验 (BOOLEAN_CHECK)

使用 'and' 进行交集判断。两个条件都必须为 True 才能通过安全网关。

反转逻辑 (INVERSION_LOGIC)

使用 'not' 来翻转当前状态。这对于切换系统开关至关重要。

logic_gate.py 
is_admin = True
is_locked = False

# 复杂条件门控
if is_admin and not is_locked:
    access_granted = True
    print("系统权限覆盖成功")
else:
    print("终端访问被拒绝")
